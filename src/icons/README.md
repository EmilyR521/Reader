# PWA icons

For the **install prompt** to appear on mobile (e.g. Chrome Android), the app needs:

- **icon-192.png** – 192×192 px  
- **icon-512.png** – 512×512 px  

Add these files in this folder. You can generate them from `../favicon.svg` using:

- [Favicon.io](https://favicon.io/favicon-converter/) (upload the SVG, download the package and copy the 192 and 512 PNGs here), or  
- [RealFaviconGenerator](https://realfavicongenerator.net/), or  
- Any image editor (export the SVG at 192×192 and 512×512 as PNG).

Name the files exactly: `icon-192.png` and `icon-512.png`.
